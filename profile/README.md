# Mastiq

Mastiq is a research-driven technology initiative focused on intelligent perception systems, accessibility, and future human-AI interaction.

We explore systems that help humans better understand and interact with the world through real-time environmental awareness, assistive intelligence, and adaptive technologies.

## Areas of Interest
- Accessibility AI
- Computer Vision
- Real-Time Perception Systems
- Human-AI Interaction
- Environmental Intelligence
- Assistive Technologies

## Current Focus
Developing perception assistants for accessibility use cases.

## Long-Term Vision
Building technologies that expand human perception and independence.
